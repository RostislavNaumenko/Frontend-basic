Css position (⊙_⊙;)

Всего существует 5 значений для свойства position в CSS

  • Static - значение и поведение элементов по умолчанию
  • Fixed - вырывает элемент из потока и фиксирует на месте, через top, bottom, left, right можно управлять положением. Элемент становится невидимым для других элементов и не влияет на scroll
  • Sticky - позиционируется в зависимости от положения прокрутки пользователя, он позиционируется relative до тех пор, пока в окне просмотра не встретится заданная позиция смещения — тогда он «застревает» на месте
  • Relative - элемент с position: relative; располагается относительно своего нормального положения.Установка свойств top, right, bottom и left элемента с относительным позиционированием приведет к тому, что он будет отрегулирован в сторону от своего нормального положения. Элемент не вырывается из потока элементов и 'сохраняет' за собой место.
  • Absolute - универсальный способ пододвинуть элемент. Элемент вырывается из потока и позиционируется относительно body или относительно своего первого отпозиционированного родителя.
